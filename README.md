# DM Pipeline

A performance dashboard application for tracking and analyzing DM pipeline metrics.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- React Router
- TanStack Query

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

```sh
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```

## Features

- DM Pipeline Performance Dashboard
- User Authentication
- Real-time Analytics
- Responsive Design
